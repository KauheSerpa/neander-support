📘 Instruções Básicas

O Neander CPU Simulator permite simular a execução de programas em uma arquitetura simples.

Funcionalidades principais:

Visualização de memória e instruções
Edição direta de valores (duplo clique)
Controle manual dos registradores (ACC, PC, RI, REM, RDM)
Execução passo a passo (1-Step)
Execução contínua (Run)
Reset da CPU

Como usar:

Clique em uma linha de memória para selecionar
Dê dois cliques para editar valores
Use os botões:
1-Step para executar uma instrução
Run para execução contínua
Reset para reiniciar
Use o botão (?) para ajuda dentro do app
🔒 Política de Privacidade

O aplicativo Neander CPU Simulator não coleta, armazena ou compartilha dados pessoais dos usuários.

Nenhuma informação é enviada para servidores externos.
Todos os dados utilizados permanecem localmente no dispositivo.

⚙️ Compatibilidade

Compatível com macOS.

📌 Observação

Este aplicativo é destinado a fins educacionais, auxiliando no aprendizado de arquitetura de computadores.
