Neander CPU Simulator

Versão 1.0

Aplicativo educacional para simulação de uma CPU baseada na arquitetura NEANDER.

📘 Instruções Básicas

  • Clique em uma linha de memória para selecionar um endereço
  • Dê dois cliques para editar valores diretamente
  • Use os controles centrais para alterar registradores
  • 1-Step: executa uma instrução por vez
  • Run: execução contínua
  • Reset: reinicia a CPU
  • Use o botão (?) para ajuda dentro do app

🔒 Política de Privacidade

O aplicativo Neander CPU Simulator não coleta, armazena ou compartilha dados pessoais.

Todos os dados permanecem localmente no dispositivo do usuário.

Nenhuma informação é enviada para servidores externos.

⚙️ Compatibilidade

Compatível com macOS.

📌 Sobre

Este aplicativo foi desenvolvido com fins educacionais para auxiliar no aprendizado de arquitetura de computadores.
